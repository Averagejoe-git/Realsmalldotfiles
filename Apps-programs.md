One BIG note I almost forgot is that NOT ALL packages come from using pacman. Arch has this little thing called the AUR, which stands for Arch User Repository. These are packages that are either made or "ported" to arch linux which is how you might get some apps like "chrome" :eye_roll"

Apps such as Obsidian require you to use the AUR since the developers don't officially support Linux systems. 

This involves the use of Yay or Paru which I had my script set up for you. You can look up the process on your own but it isn't too interesting so just take it as a note

I can list out some of my favorite Apps that may be useful to you for your work here, although you may already have 1 or 2 of them already.

***List:*** 
	1. **Vesktop** 
		Vesktop is essentially a Linux supported layer placed on top of Vencord (aka discord) which is supposed to make the discord environment much friendlier to Linux systems since Discord doesn't really care about Linux. 
	2. **Obsidian** 
		Obsidian is an extremely popular note-taking app that is supported almost everywhere besides maybe IOS. Its very useful for taking notes and for managing them as you can apply tags, references to other notes, and even create a cool little web graph (youll see if you click on the second little icon on the top left that has 3 circles)
	3. **LibreOffice**
		LibreOffice is honestly one of my favorite tools. Its essentially the free and open source version of Microsoft's tools such as Docs/words, slideshow, and spreadsheet. It also follows the GTK theme (your orange and black theme) so it doesn't blast you with light. 
	4. Qalculate!
		A tool I sometimes use for school that I prefer over using desmos. Its a calculator with a wide variety of options for math problems. 
	5. **Sober** 
		Sober is roblox but translated through an Android version/layer. Most people say it improves performance but its up to you to decide to be honest. You download it through flatpak but I reckon they'll give you the command for it if you just look up org.vinegarhq.sober (Don't look it up as the site but just the name)
	6. **VLC or MPV Media Player**
		Both of these are lightweight tools used to play videos or audio. Thats pretty much it. Although I will say I prefer MPV for music. 
	7. **Wine**
		Wine is simply a translation layer for apps or programs that don't natively support Linux. Some games may require Wine as well. Its a little bit annoying and confusing to set up though. 
	8. **Btop + Nvtop**
		Both of these are tools used to monitor system resource usage. Btop shows everything from memory, gpu, cpu, cpu temps, etc. While Nvtop specifically shows an in-detail graph on the GPU. You probably won't need it, but its still just as useful as having the task manager on Windows. 
	9. **ZapZap** 
		In the simplest way, ZapZap is just a Whatsapp program for Linux. Its open source and I believe available in the AUR. 
	10. **Gurk**
		Although possibly not useful to you, gurk is a TLI (Terminal line interface) app for Signal. It offers better security and stores all files sent locally (on your device only). 

One last thing, there are CLI, TLI, and GUI. CLI means command line interface, meaning its controlled by commands in the terminal. TLI stands for terminal line interface, meaning that it doesn't run off only commands but it doesn't have a separate app, it runs off of the terminal. (For example, run "bluetoothctl") 

GUI stands for Graphical User Interface, which is basically the normal apps you see everyday, they have their own designs (such as this one).
